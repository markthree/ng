# g

simplify git operations

## Usage

### install

```shell
deno install --allow-run --allow-read --allow-env -rfn g https://deno.land/x/m3g/mod.ts
```

### cli

```shell
g # git add .

g c "foo" # git commit -m "foo"

g p # git push

g -h # help information
```
